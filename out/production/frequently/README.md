## String、StringBuffer、StringBuilder异同：
    String:
        1.不可变的字符序列
        2.底层使用char[]存储
    StringBuffer:
        1.可变的字符序列
        2.线程安全，效率低。
        3.底层使用char[]存储
        --------------------------
        
    StringBuilder:
        1.可变的字符序列
        2.线程不安全，效率高。
        3.底层使用char[]存储
